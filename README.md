# Ustav Hub 
*A Flutter-based Event Management Application*

# Project Overview
**Ustav Hub** is a Flutter-based mobile application developed as part of my internship/final year project.  
The app simplifies **event management** by providing separate modules for **Users** and **Admins**.

Users can browse events and send participation requests, while Admins can manage events, approve requests, and send notifications.

# Objectives
- To digitize and simplify event management
- To provide a smooth user experience using Flutter
- To implement role-based access (User/Admin)
- To gain hands-on experience with real-world app development

# Tech Stack
- **Frontend:** Flutter (Dart)
- **UI Design:** Material UI, Google Fonts (Poppins)
- **Backend / Database:** Firebase (Auth / Firestore) *(if used)*
- **Tools:** Android Studio, Git, GitHub

# User Roles & Features

# User Module
- View available events
- Select event preferences (food, theme, etc.)
- Send event participation requests
- Receive notifications on approval/rejection

# Admin Module
- Create and manage events
- View user requests
- Approve or reject event participation
- Send notifications to users
